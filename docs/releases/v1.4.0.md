# Release v1.4.0: Chaos Engine & Refactoring

**Date:** 2026-02-09
**Developer:** Manuel Vasquez

## Overview
This release introduces controlled data anomalies (Chaos) and reorganizes the project structure.

## Key Changes
1.  **Chaos Engine**:
    - Adds `ChaosEngine` to inject Nulls, Duplicates, and Outliers.
    - Configuration via `defaults.yaml` (Profiles: low/medium/high).
    - Affected Domains: Orders (Sales), Customers, Products.

2.  **Refactoring**:
    - Moved `src/yupay/domains/inventory` -> `src/yupay/domains/sales/inventory`.
    - Aligns inventory as a dependency of the Sales domain.

## Verification
- Run `yupay generate sales` to test.
- Use `python src/tools/verify_chaos.py` to audit anomaly rates.
