# Release v1.3.0: Advanced Seasonality

**Date:** 2026-02-09
**Status:** Stable

## Overview
This release implements a **Composite Time Engine** to simulate realistic temporal patterns in sales volume.

## Key Features
*   **Composite Signal**: Demand = Base * (Trend * Seasonality * Weekly * Shock).
*   **Linear Trend**: Annual growth simulation (+0.5% monthly).
*   **Event Calendar**: Native support for Peruvian holidays (Fiestas Patrias, Christmas) and dynamic events (Mother's Day).
*   **Weekly Patterns**: Configurable weights for days of the week (e.g., Sat > Mon).

## Technical Details
*   **Refactor**: `TimeProfile` and `TimeEngine` completely rewritten.
*   **Configuration**: Added `Retail Peru` profile.
